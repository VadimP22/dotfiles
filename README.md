# My dotfiles
- Helix - code editor
- Hyprland - window manager
- Waybar - status bar
- Mako - notifications daemon
- Swaylock Effects - lock screen
- Wlogout - session management gui
- Firefox Theme: https://github.com/datguypiko/Firefox-Mod-Blur/tree/master
